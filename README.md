# MAR-Conductance
Code to compute electornic transport between two superconducting electrodes and a magnetic impurity.

 Compilation: please execute "make" on your command line (Linux or Unix machine)
 Edit MAR.in and change the parameters that are explained in the provide MAR.in for your problem
 Execute MAR_02.out
 the second output file contains the applied bias in mV and the Conductance in G_0; the first file contains the bias and the electronic current
